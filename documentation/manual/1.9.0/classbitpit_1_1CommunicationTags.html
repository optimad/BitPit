int bitpit::CommunicationTags::generate ( MPI_Comm  communicator)

Generates a unique tag.

If the trash is empty a new tag is generated, otherwise a tag taken from the trash is recycled.

The tag is shared among the processes that belong to the specified communicator.

Parameters
communicatoris the MPI communicator
