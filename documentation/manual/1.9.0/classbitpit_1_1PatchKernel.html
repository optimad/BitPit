Extracts the neighbours of the specified cell for the given local vertex.

Cells that has only a vertex in common are considered neighbours only if there are other cells "connecting" them. 

              .-----.                   .-----.
              |     |                   |     |
            V | A1  |                 V | A2  |

        |     |                   |     |     |
        | B1  |                   | B2  | C2  |
