void bitpit::SystemSparseMatrixAssembler::getRowData ( long  rowIndex,
ConstProxyVector< long > *  pattern,
ConstProxyVector< double > *  values 
) const
overridevirtual

Get the data of the specified row.

If the matrix is a block matrix (i.e., the block size is greater than one), the pattern defines the global ids of the block columns of the row, where a block column is defined as a group of blockSize matrix columns.

If the matrix is a block matrix (i.e., the block size is greater than one), the values contain of all the elements of a block row, where a block column is defined as a group of blockSize matrix columns. The values are returned as a row-oriented logically two-dimensional array of values.

Parameters
rowIndexis the index of the row in the assembler
patternon output will contain the pattern of the specified (block) row
valueson output will contain the values of the specified (block) row. If the block size is greater than one, values will be stored in a logically two-dimensional array that uses a col-major order
